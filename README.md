## Reacciones cobra
Cada Modelo incluido tiene reacciones y dentro de las reaccciones tiene metabolito y genes.
Cada modelo tiene un objetivo.
```
Modelo
+--Reacccion
   +--Metabolito
   +--Genes
+--Objetivo
```
